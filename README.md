###群宗旨

**东西是写出来的，不是用眼睛看出来的，只有动手实践，才能更加熟练，才能提升自己。你遇到过下面的情景吗？**

- 总想学点东西，却不知道学什么
- 刚学完一块知识，想实践一下，却不知道写什么
- 写代码的时候知道思路，但总是无从下手
- 写一些简单样式的时候也要去百度
- 感觉自己会的东西不系统，左一块，右一块
- 想动手写一些项目，自己却没有想法
- 缺乏团队协作开发经验

如果你遇到过上面的问题，证明你的实践远远不够。所以你该多动手写写代码了。
